Now it's time to start testing, I will do a 1-way ANOVA of the regions with the observations being the sightings from each state. Because states with higher populations are likely to experience more sightings as well as more reports per sighting, I will normalize each state's number of sightings based on the state's population as a proportion of the US populaiton like this: $$sightings_{norm}(state)=\frac{sightings(state)}{.000001\cdot statePop}$$ To make this easier, I got a CSV from the US Census Bureau. I skipped the first row because it had data I did not need, and the last row because it was for Puerto Rico I made the second row the header of the data frame (indexing starts at 0 in Python). Because the Census data used the full state names, I created a CSV with the abbreviations sorted in the same order as the alphabatized states and will replace the old one:

So, after running a one way $\textbf{ANOVA}$ and permutation F-Test and failing to find any significant differences in the population-normalized regional distributions, I decided to do pairwise K.S. tests on all the pairwise combinations, finding 3 groups that seemed to be different, I did a permutation pairwise K.S. test and found even more difference. Furthermore, a look at the boxplots and histograms gives a picture of a lot of overlap, with the potential for some of the groups to be distinct. I was going to run some of these tests as well as just tests on generating long lists of combinations in R as well, but This data was quite messy and incomplete for the research I was doing, so I spent a lot of time cleaning and formatting it, as well as some time learning how to use Python's MatPlotLib library to make my graphs. I think that since the Northwest showed up the most in my pairwise tests ans pairwise permutation tests, that it might be slightly more distinct fromt he other regions in sightings. Looking at the boxplots and histograms reinforces this hypothesis and lets me know the direction of the relationship. The Northwest seemed to experience more sightings per capita than any other region.
